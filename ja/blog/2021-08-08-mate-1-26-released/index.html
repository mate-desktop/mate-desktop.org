MATE 1.26 released

Wolfgang Ulbrich

2021-08-10 09:00

This release might have taken a little longer than usual but now after 17 month of development we are very pleased to release MATE 1.26. A big thank you to all contributors who helped to make this happen.

What’s new in MATE 1.26

The theme for this release has been adding new functionality to the MATE Desktop while maintaining the look and feel that we all know and love. While all the added features are surely quite exciting we also did not forget to do tons of bugfixing, modernising the code base and optimizing the performance.

The headline changes in MATE Desktop 1.26 are:

  • We added Wayland support for Atril, System Monitor, Pluma, Terminal and other components of the Desktop.

  • Pluma was subject to massive improvements.

  • There is a new mini map which gives you instant overview over the content.
  • The new grid background pattern gives you the feel to write on paper.
  • The sort plugin now supports undo actions.
  • We added show/hide line-numbers shortcut (Ctrl+Y).

  • The preferences dialog needed a redesign to catch up with all the new features.

  • Together with the brand new pluma-plugins package Pluma can be turned into a feature-rich IDE, with support for Bracket completion, Comment/Uncomment code, Built-in terminal and Wordcompletion.

  • The Calculator is better in every single way.

  • Port to GNU MPFR/MPC Library, which means more precision, faster computation, more supported functions.
  • We finally added an optional history to show recent calculations.
  • Calculator is now resizeable if history is activated.
  • Vastly improved prime factorization speed.

  • Much faster modular arithmetic evaluation (modular exponentiation ability).

  • The Control Center features an improved Windows preferences dialog with more options. In the Display dialog we added an option for window scaling.

  • We finally added a Do not Disturb Applet.

  • Speaking of Applets, the window-list applet has a new option to disable mouse scrolling. The window-list thumbnails are now crisp and clear as they are rendered as cairo surfaces.

  • The netspeed applets shows more information by default and gained netlink support.

  • …

  • Translations are updated. Thank you to our team of translators!

The various components of MATE Desktop 1.26 are available for download here:

  • https://pub.mate-desktop.org/releases/1.26/
