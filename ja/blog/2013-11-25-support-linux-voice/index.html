MATE チームはあなたが Linux Voice on indiegogo へ出かけて、オープンソースのユーザーでありファンでもあるあなたがなぜ Linux Voice を支援した方がいいのかを調べることをお勧めします。このプロジェクトを支持する理由は、以下のとおりです。

  • Linux Format 誌の編集チームの多数によって結成されました。
  • Linux Voice 誌の利益の50%はフリーソフトウェアと Linux コミュニティに還元されます。
  • Linux Voice の内容は、9ヶ月後に CC-BY-SA ライセンスによって無料で公開されます。
  • Linux Voice チームの 1 人は、プロジェクトへの資金提供が成功した場合、MATE を支援することに興味を持っています。

Linux Voice に支援を！

このキャンペーンの進捗状況を詳しく知るには、Linux Voice のウェブサイトにアクセスしてください。
